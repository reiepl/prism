# PRISM

**Personal Relationship Intelligence System for Metaphysics**

PRISM is a privacy-first, offline-first personal relationship intelligence platform that combines traditional Chinese metaphysics (BaZi, Tong Shu, and related systems) with modern software engineering.

The application allows users to build a structured database of people, generate BaZi blueprints, analyse relationships, and perform Tong Shu date selection, all without requiring cloud services.

---

## Vision

To become the most comprehensive personal relationship intelligence platform built upon Chinese metaphysics while remaining completely private, explainable and user-controlled.

---

## Core Principles

- Privacy First
- Offline First
- Mobile First
- Explainable Analysis
- Modular Architecture
- AI Assisted Development
- Domain Driven Design
