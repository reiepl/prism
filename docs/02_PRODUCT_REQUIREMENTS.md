Scope:

# Product Requirements

## Product Name

PRISM

Personal Relationship Intelligence System

---

## Target Users

Primary

Individuals interested in Chinese metaphysics.

Professional BaZi consultants.

Relationship consultants.

Date selection practitioners.

---

## Problem Statement

Existing metaphysics software suffers from:

- Poor user interface
- Poor data organisation
- Limited explainability
- Cloud dependency
- Outdated technology
- Weak relationship management

PRISM aims to solve these issues.
